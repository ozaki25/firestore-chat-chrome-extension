## 使い方

### セットアップ

```bash
git clone https://github.com/ozaki25/firestore-chat-chrome-extension
```

- 拡張機能を開く
    - Chromeを開く -> その他ツール -> 拡張機能
- 拡張機能のインストール
    - パッケージ化されてない拡張機能を読み込む -> cloneしたフォルダを選ぶ
- キー情報の設定
    - 詳細 -> 拡張機能のオプション -> APIKeyとProjectIDを入力
